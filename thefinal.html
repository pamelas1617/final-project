Final Project Overview

This website was created for GreenTech Solutions, a company focused on providing eco-friendly technology solutions. The purpose of the website is to inform users about sustainable technologies, showcase products, and allow users to contact the company.

Target Audience

The website is designed for environmentally conscious consumers, businesses looking to reduce energy usage, and individuals interested in sustainable living.

Technologies Used

  • HTML5 (structure)
  • CSS3 (responsive design and styling)
  • JavaScript (form validation and product search)

Accessibility Features

  • Proper form labels for screen readers
  • Keyboard-friendly navigation
  • High contrast eco-friendly color scheme
  • Responsive layout for all screen sizes

Website Features

  • Navigation bar on every page
  • Product search functionality
  • Contact form with validation
  • Responsive design for mobile and desktop
  • Informational content sections
