# GreenTech Solutions Website

## Project Description
The website provides information about green technology, showcases products, and allows users to contact the company.

The goal of this project is to demonstrate skills in HTML, CSS, JavaScript, responsive web design, and accessibility best practices.

## Pages Included
- Home Page
- About Page
- Products Page
- Resources Page
- Contact Page
- Final Project Page

## Technologies Used
- HTML5 
- CSS3 
- JavaScript 
- Responsive Web Design 

## How to Run the Project
1. Click on url
2. Click on any button on the navigation bar to explore content
